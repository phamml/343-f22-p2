  2. App functionality: Film Finding is a basic movie searching app that allows users to input a keyword or a specific movie title and then generates results based on the input provided. The results generate a list of movies that match the provided input displaying the movie title, movie poster img, release date, user rating, list of providers that movie is available on, and a brief overview of the movie. Once results are displayed, users will be allowed to filter results by release date as well as sort alphabetically by movie title. Displays max 40 results due to time complexity issues and to make sure app is efficient. Also, there are a limited number of calls to API. Would love to improve on this with more time to allow app to show all reults.
  3. This app satifies the assignment's requirements by:
    1. App is deployed publicly with github pages
    2. App is responsive with media queries to control how many columns are present for larger and smaller screens
    3. App uses at least two different APIs: one for getting movie title, movie poster img, release date, and movie overview and one for getting user rating and available watch providers
    4. App has search bar that lets users search for movies with keywords or titles
    5. App displays results from search displaying movie title, movie poster img, release date, user rating, list of providers that movie is available on, and a brief overview of the movie.
    6. App allows users to filter results by release date and sort alphabetically by movie title
